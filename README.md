# RemindersApp

Приложение с использованием фреймворков .Net 3.1.100 и Angular 8.0.3, предназначенное для отправки уведомлений пользователю.

## Установка и запуск

Открыть командную сторку в корне проекта.

Для запуска бэка 
```
cd RemindersApp
dotnet run
```
Для запуска фронта 
```
cd RemindersApp/ClientApp
npm install
npm run dev
```
